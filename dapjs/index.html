    Using Pitaya-Link with DAP.js¶

    Introduction¶

    DAP.js is a JavaScript interface to CMSIS-DAP, enabling access to Arm Microcontrollers using Node.js or in the browser using WebUSB.

    This page demonstrates the flash exmaple using DAP.js.

    Refer to the DAP.js API Documentation for more information.
